# 🦖 班级电子宠物 (Class Pet)

**“班级电子宠物平台”** 是一款结合了游戏化元素（Gamification）的现代化班级管理与激励工具。通过为每位学生分配一个虚拟的“电子宠物”，将现实中枯燥的班级表现（如纪律、学习、劳动等）转化为可视化的成长经验值（EXP）。

基于多邻国（Duolingo）风格的果冻感与微质感 UI 设计，本项目旨在为师生提供极强的正向情绪价值，用“养成体验”激发学生的内驱力和集体荣誉感。

## ✨ 项目简介

在这个平台上，每个学生都拥有一颗属于自己的宠物蛋。当学生在现实中表现良好（如主动回答问题、帮助同学）时，教师会为其加分；反之则扣分。所有的积分不仅影响个人数据，更会直接反馈在大屏幕上，推动宠物等级提升和形态进化。

本系统采用 React + Vite + Tailwind CSS + Framer Motion 构建前端，并集成 Firebase (Firestore) 实现极其流畅的多端实时数据同步。

## 🚀 1.0 版本已实现功能

目前 1.0 基础版本已上线，主要包含以下核心功能：

### 🚪 统一入口门户 (Portal)
清晰的首屏导航，将用户分流至“教师工作台”、“班级大屏”及“学生查询”三个不同场景。

### 👩‍🏫 教师工作台 (Teacher Panel)
*   **名单管理**：支持快速添加新生名册并自动分配初始宠物蛋。
*   **设置与安全**：初始的教师安全访问验证与班级特征码（Class PIN）设置。
*   **快捷行为记分**：内建正向（如：作业优秀 +5）与负向（如：迟到 -3）行为标签，支持快速一键为单一或多个学生加扣分。
*   **实时同步反馈**：教师端的操作，班级大屏将会进行近乎无延迟的实时同步更新。

### 📺 班级大屏展示页 (Big Screen)
*   **全景视图**：全响应式网格布局，适合在教室多媒体屏幕长期展示全班的宠物状态。
*   **动态动画**：采用弹簧力学动画 (Spring Physics)，包含宠物形象呼吸效果与进度条加载。
*   **实时更新**：学生经验值一旦变动，大屏信息自动平滑刷新，无需手动干预。

### 🧑‍🎓 学生查询页 (Student Query)
*   **独立查询**：输入班级特征码（Class PIN），即可在家长手机/平板等受限设备上查看个人详情，无越权越界风险。
*   **成长账单**：清晰列出近期的加扣分历史原因及具体时间，方便家长监督与见证点滴进步。

### 🎮 基础成长系统
*   **升级防挫折机制**：包含完备的等级计算逻辑。扣分仅扣除当前等级内的 EXP，即使用户经验扣减至负数，也绝不会导致宠物“降级”，最大程度保护学生自尊心和动力。

## 🔮 未来版本规划 (Roadmap)

系统将不断迭代演进，以下功能正在开发并计划在未来上线：

*   **🎙️ AI 萌宠对话助手 (AI Pet Chat)**
    - 集成大语言模型 (Gemini / Deepseek / ChatGPT 等)。
    - 学生点击宠物即可互动对话。宠物将基于内置的 **K-12 黄金护卫安全 Prompt** 进行回复，结合该学生近期的真实加减分记录，用温暖鼓励的语气指导孩子学习与成长。
    - 所有模型 API Key 配置仅限于本地存储（LocalStorage），确保 100% 用户隐私安全。
*   **🎁 盲盒进化系统 (Blind Box Evolution)**
    - 升级不仅增加数字，更能逐渐从“盲盒池”内抽取特效挂件（如：天使翅膀、皇冠、光环、炫彩特效）。
    - 宠物外形通过多层挂件（背饰/头饰/底部特效等）动态叠加，增强学生养成满足感与收集欲。
*   **🎯 O2O 现实奖励闭环体系**
    - 后台将集成“里程碑提醒”。每当学生达到设定的特殊等级（如 Lv.10, Lv.20），系统向教师提示并核销相应的班级特权（例如：“免写一次作业”、“换一次心仪座位”）。
*   **🔁 赛季轮回与图鉴收集 (Season Reset)**
    - 为了避免长时间累计导致学渣失去冲劲、学霸失去目标，将推出按学期结算的赛季制。
    - 赛季结束时当前宠物可收入“名人堂图鉴”，新的一学期自动转生为全新萌蛋，实现机制焕新。
*   **📊 数据仪表盘与互动动画增强**
    - 增加更详细的大屏 Top 榜单展示。
    - 在发生加减分变更时，大屏中该学生卡片自动高亮置顶，并配有视觉强烈的“粒子爆炸金币喷出”等交互动画特效。

## 🛠️ 技术栈与部署说明

*   **前端**：React 18 + TypeScript + Vite
*   **UI 框架**：Tailwind CSS (拓展果冻色调) + Lucide React 图标
*   **动效引擎**：Framer Motion (弹簧物理反馈)
*   **后端 & 实时通讯**：Firebase Authentication + Firestore

### 在本地启动

1.  克隆本仓库到本地。
2.  执行 `npm install` 或 `yarn` 安装依赖包。
3.  在项目根目录下完成 Firebase 的初始化 (`firebase-applet-config.json`)。
4.  执行 `npm run dev` 启动开发服务器。
